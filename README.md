# Turbo RMF
Requirements
- Python 3.6
- Django Python 2.0.1
- Pywinrm 3.0
- Django Rest Framework v3


# Install Resources and Run
- pip install Django==2.0.1
- pip install djangorestframework
- pip install pywinrm
- cd ${PROJECT ROOT}
- python manage.py runserver
